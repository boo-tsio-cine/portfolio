Portfolio React JS & Laravel

Ceci est un portfolio réalisé avec React JS pour le frontend et Laravel pour le backend.

Instructions d’installation

Les bases de données sont gérées par Laravel et contiennent :

La liste des projets réalisés

Les études effectuées

Les réalisations

Importez la base de données depuis backends/portfolio.sql dans votre serveur MySQL.

Lancez le serveur Laravel :

Ouvrez le terminal dans le dossier backends

Tapez la commande :

php artisan serve


Lancez le frontend React :

Ouvrez le terminal dans le dossier frontend

Tapez la commande :

npm start


Créez un compte admin pour accéder au CRUD et voir toutes les fonctionnalités de Laravel.

Une fois le frontend lancé, vous pouvez consulter le portfolio et naviguer à travers les projets.
